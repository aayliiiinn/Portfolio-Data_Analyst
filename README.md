# Portfolio-Data_Analyst
Ce portfolio vise à démontrer :

la capacité à analyser des besoins métier ;

la structuration et la gouvernance de la donnée ;

la production de livrables clairs et exploitables ;

la maîtrise d’outils data (Python, SQL, Power BI, Excel, KNIME) ;

la communication et la vulgarisation de résultats data.
